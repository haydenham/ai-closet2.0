# Quiz Item Images

This folder contains the static images used in the style quiz.

## Structure
- 50 images total (5 question types Ã— 10 style categories)
- All images for female quiz
- Public static assets served by FastAPI

## Naming Convention
Images should be named: `{style}_{type}.jpg`

Examples:
- `bohemian_pants.jpg`
- `streetwear_shirt.jpg`
- `classic_shorts.jpg`
- `edgy_overlayer.jpg`
- `minimalist_shoes.jpg`

## Image Requirements
- Format: JPG, PNG, or WebP
- Recommended size: 400x600px (portrait)
- Max file size: 2MB per image
- Should clearly represent the style category
